Building a symptom-based disease diagnosis web app with Flask and machine learning

People often experience symptoms and want quick answers. Differentiating between diseases is hard when symptoms overlap. This project tackles that problem head-on — using a trained SVM classifier and a simple web interface to give users instant, personalised health information based on what they're feeling.

1
Symptom input

Users type comma-separated symptoms into the interface. The app supports 132 distinct symptoms.

2
SVM prediction

Symptoms are encoded into a binary vector and passed to a trained Support Vector Classifier, which predicts from 41 diseases.

3
Full health plan

The predicted disease is looked up across CSV datasets to return medications, diet advice, precautions, and workouts.
